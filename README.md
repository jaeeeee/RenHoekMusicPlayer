# Ren Höek's Music Player (RimWorld Mod)

An in-game music player widget with context-aware playback options (Day / Night / Winter / Combat), shuffle, seek, and keybinds.

## Features
- Context-aware playback (optional): Day / Night / Winter / Combat
- Shuffle + Previous/Next within the current context
- Seek bar + time display
- Silence between songs (min/max) with overrides
- Fade speed options
- Draggable/resizable widget + compact mode
- Search/filter song list

## Requirements
- Harmony (load before this mod)

## Install (GitHub Release)
1. Download the latest zip from Releases on this repo.
2. Extract the RenHoekMusicPlayer folder into your RimWorld Mods folder.
3. Enable it in RimWorld (load after Harmony).

## Keybinds
Options → Keyboard → Ren Höek's Music Player

## Settings
Options → Mod Settings → Ren Höek's Music Player
